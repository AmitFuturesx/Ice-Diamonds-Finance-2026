# Soft Ice Diamonds - מערכת ניהול פיננסי לעסק תכשיטים ויהלומים 💎

ברוכים הבאים ל-**Ice Diamonds** - פלטפורמת ניהול פיננסי חכמה, מלוטשת ויוקרתית המותאמת אישית לצרכי ענף התכשיטים והיהלומים ובנויה בסטנדרט העיצובי של מערכות "חשבונית ירוקה" (Morning) המוכרות בישראל.

המערכת פועלת בממשק עברית מלא (RTL), מותאמת באופן מושלם למובייל (Mobile-First) וכוללת סינכרון בזמן אמת של כלל המדדים הכספיים.

---

## ✨ תכונות מרכזיות במערכת

1. **לוח בקרה (Dashboard):**
   * תצוגת מאזן חודשי נטו ברווחיות או גירעון (באור ירוק או אדום).
   * פילוח הכנסות לפי שערי סליקה (מקסימוס, UPS, מזומן, העברה בנקאית, קארדקום).
   * פילוח קבוצות הוצאות (קבועות, משתנות, הלוואות) ביחס אחוזים שבועי, ומדד יעילות תזרים.
   
2. **הוצאות קבועות (Fixed Expenses):**
   * ניהול שכירות, ביטוחים, ארנונה ותקשורת.
   * סירוס ותמרון תגי סטטוס תשלום מהירים (שולם / שולם חלקית / לא שולם) בלחיצת כפתור אחת מהירה היישר מהטבלה.
   * הוספה, עריכה ומחיקה מהירה של שורות.

3. **הוצאות משתנות (Variable Expenses):**
   * מעקב רכישת זהב/כסף/יהלומים עם עמודת תאריך, הערות וסכומים במטבע שח.

4. **מעקב חובות והלוואות (Debts Tracker):**
   * טבלה חכמה עם החזרים חודשיים, יתרת חוב וקרן מקורית.
   * מד פירעון חזותי המראה באחוזים את התקדמות סילוק הקרן עבור כל הלוואה בנפרד וברמת המקרו.

5. **דוח הכנסות (Income Streams):**
   * **מטריצת פילוח שבועי:** טבלה המרכזת את כלל חמשת שעריי ההכנסה ומחלקת אותם לפי שבועות (שבוע 1 עד 4) עם סיכומי ביניים שבועיים אוטומטיים.

6. **שליחויות והפצה (Deliveries):**
   * מעקב אחרי קווי משלוח מבוטחים, כולל שערי מע״מ (17% VAT) המחושבים אוטומטית בצורה חיונית לדוחות רואי חשבון.

7. **סנכרון חודשים חכם:**
   * בורר חודשים דביק בראש המסך המשפיע מיידית על כל הדוחות והנתונים.
   * אפשרות להוספת חודשים חדשים למאגר הנתונים דרך הממשק בלחיצת כפתור.

## 🚀 הוראות התקנה והרצה מקומית

המערכת פותחה באמצעות **React 19, TypeScript ו-Tailwind CSS** ותומכת בחיבור ישיר ל-**Supabase** או בסנכרון פנימי אוטומטי (LocalStorage Fallback) לעבודה מיידית במצב Sandbox ללא הגדרות קודמות!

### שלב 1: התקנת חבילות
יש להריץ את הפקודה הבאה בטרמינל להתקנת התלויות:
```bash
npm install
```

### שלב 2: הגדרת משתני סביבה (Supabase Setup)
כדי להתחבר למאגר נתונים חי ב-Supabase, העתק את הקובץ `.env.local.example` לקובץ חדש בשם `.env.local`:
```bash
cp .env.local.example .env.local
```
והזן את פרטי הגישה של הפרויקט שלך מתוך לשונית ה-API ב-Supabase:
```env
VITE_SUPABASE_URL="https://YOUR_PROJECT_ID.supabase.co"
VITE_SUPABASE_ANON_KEY="eyJhbGciOi..."
```

> **מצב דמו פעיל (Sandbox Mode):** אם משתני הסביבה לא יוזנו (או יושארו כברירת מחדל), האפליקציה תפעל ב-Local Demo Mode המאחסן את הנתונים בדפדפן, ומגיע מוכן וטעון מראש עם הנתונים המלאים של **אפריל 2026**!

## 🗄️ הקמת הטבלאות ב-Supabase (PostgreSQL)

כדי להקים את מאגר הנתונים ב-Supabase, פתח את ה-**SQL Editor** בלוח הבקרה של Supabase, והרץ את תוכן הקובץ המצורף ב-`/supabase/seed.sql`.

פקודה זו תבצע:
1. יצירת הטבלאות הנדרשות: `months`, `fixed_expenses`, `variable_expenses`, `debts`, `income_entries` ו-`deliveries`.
2. הגדרת מפתחות זרים לעדכון ומחיקה מדורגים (`ON DELETE CASCADE`).
3. הפעלת מנגנוני אבטחה של Supabase Row Level Security (RLS) המאפשרת גישה בלעדית למשתמשים מחוברים.
4. הזנת נתונים ראשוניים (Seed Data) של חודש **אפריל 2026** כדי שהפלטפורמה תיראה חיה ועשירה מהשנייה הראשונה!
